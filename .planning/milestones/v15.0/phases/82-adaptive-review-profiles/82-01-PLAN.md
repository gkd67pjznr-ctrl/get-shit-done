Implement the following:

**Constants:**

```js
const MIN_CORRECTIONS = 10;
```

**`generateReviewProfile(options)`** function:

1. Resolve `cwd = (options && options.cwd) ? options.cwd : process.cwd()`.

2. Wrap entire body in try/catch — on any error, return `null` silently.

3. Call `readCorrections({ status: 'active' }, { cwd })` imported from `./write-correction.cjs`. This returns an array of correction objects.

4. Filter corrections to only those whose `diagnosis_category` is a key in `CATEGORY_SKILL_MAP` from analyze-patterns.cjs — BUT do not import analyze-patterns.cjs. Instead, define the same 14-category set inline as a `const VALID_CATEGORIES = new Set([...])` mirroring `VALID_CATEGORIES` in write-correction.cjs.

   Check the actual valid category set by reading write-correction.cjs — it exports `VALID_CATEGORIES` internally but does not export it from module.exports. Reproduce the set inline in review-profile.cjs. The 14 categories are:
   ```
   'code.wrong_pattern', 'code.missing_context', 'code.stale_knowledge',
   'code.over_engineering', 'code.under_engineering', 'code.style_mismatch',
   'code.scope_drift', 'process.planning_error', 'process.research_gap',
   'process.implementation_bug', 'process.integration_miss',
   'process.convention_violation', 'process.requirement_misread', 'process.regression'
   ```

5. Count active corrections per known category:
   ```js
   const counts = {};
   for (const c of corrections) {
     const cat = c.diagnosis_category;
     if (!VALID_CATEGORIES.has(cat)) continue;
     counts[cat] = (counts[cat] || 0) + 1;
   }
   ```

6. Compute `sampleSize = corrections.filter(c => VALID_CATEGORIES.has(c.diagnosis_category)).length`.

7. Check minimum guard: if `sampleSize < MIN_CORRECTIONS`, return `null`.

8. Compute normalized weights:
   ```js
   const weights = {};
   for (const [cat, count] of Object.entries(counts)) {
     weights[cat] = count / sampleSize;
   }
   ```

9. Build the profile document:
   ```js
   const profile = {
     generated_at: new Date().toISOString(),
     sample_size: sampleSize,
     min_corrections_required: MIN_CORRECTIONS,
     weights,
   };
   ```

10. Write atomically to `.planning/patterns/review-profile.json`:
    ```js
    const patternsDir = path.join(cwd, '.planning', 'patterns');
    fs.mkdirSync(patternsDir, { recursive: true });
    const profilePath = path.join(patternsDir, 'review-profile.json');
    const tmpPath = profilePath + '.tmp';
    fs.writeFileSync(tmpPath, JSON.stringify(profile, null, 2));
    fs.renameSync(tmpPath, profilePath);
    ```

11. Return `profile`.

**module.exports:**
```js
module.exports = { generateReviewProfile };
```

**CLI block** (for direct invocation):
```js
if (require.main === module) {
  const cwd = process.argv[2] || process.cwd();
  const result = generateReviewProfile({ cwd });
  process.stdout.write(JSON.stringify(result) + '\n');
  process.exit(0);
}
```

function makeCorrections(category, count, status = 'active') {

    diagnosis_category: category,
    status,
    timestamp: new Date().toISOString(),
    correction_to: `fix ${i}`,
    scope: 'project',

}
