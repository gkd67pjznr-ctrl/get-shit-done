# Plan 33-01: Wire analyze-patterns.cjs to SessionEnd Hook

## Goal

After every session ends, `analyze-patterns.cjs` runs automatically and writes a watermark timestamp to `scan-state.json`. No manual invocation required.

## Context

- `analyze-patterns.cjs` is at `.claude/hooks/lib/analyze-patterns.cjs` and is a fully working CommonJS module that exports `analyzePatterns({ cwd })`.
- `scan-state.json` exists at `.planning/patterns/scan-state.json` and currently contains `{}` — no watermark.
- `analyzePatterns()` already updates `suggestions.json`'s `metadata.last_analyzed_at` watermark, but `scan-state.json` is a separate file that needs its own watermark for external observability.
- The SessionEnd hook array in `.claude/settings.json` currently has `gsd-save-work-state.js` and `gsd-snapshot-session.js`. A new entry must be appended.

      ```js
      // SessionEnd hook: runs pattern analysis and writes watermark to scan-state.json.
      // Silent on errors -- never blocks session end.

      const path = require('path');
      const fs = require('fs');

      try {
        const cwd = process.cwd();
        const { analyzePatterns } = require('./lib/analyze-patterns.cjs');

        const result = analyzePatterns({ cwd });

        // Write watermark to scan-state.json regardless of whether new suggestions were written.
        // This confirms analysis ran even when no new patterns met thresholds.
        const scanStatePath = path.join(cwd, '.planning', 'patterns', 'scan-state.json');
        let scanState = {};
        try {
          const raw = fs.readFileSync(scanStatePath, 'utf-8');
          scanState = JSON.parse(raw);
        } catch (e) {
          // File missing or invalid JSON -- start fresh
        }

        scanState.last_analyzed_at = new Date().toISOString();
        scanState.last_analysis_result = result;

        const tmpPath = scanStatePath + '.tmp';
        fs.writeFileSync(tmpPath, JSON.stringify(scanState, null, 2));
        fs.renameSync(tmpPath, scanStatePath);
      } catch (e) {
        // Silent failure -- never block session end
      }
      ```
